Positioning Content on the Home Page

{{ page.description }}

Several objects in the configuration file have parameters which allow you to specify where the object should appear on the page. For example, static content, speakers, tracks and sponsors all have the ability to be repositioned on the page. Objects that can be positioned will have the following attributes in the configuration file:

  • insert_target - A CSS selector (a simple class or ID) that will be used as the starting point for where the given object is rendered
  • insert_mode - How the content is inserted
    • after
    • before
    • append
    • prepend

Determining the target

The insert_target parameter expects a CSS selector (using a simple class or ID format). The element matched by this selector will be used as the starting point for where the given object is rendered. The best way to determine the CSS selector or ID to use is to browse to the virtual event homepage and then use the browser developer tools to inspect the elements in the page.

You will notice that the main sections of the homepage are organized using <div> tags.

Choosing the insert mode

After determining where in the hierarchy to insert the object, the insert_mode attribute will determine how the object is inserted in the hierarchy.

  • after - Puts the object after the specified div at the same level
  • before - Puts the object before the specified div at the same level
  • append - Puts the object inside the specified div as the last child
  • prepend - Puts the object inside the specified div as the first child

Object positioning example

The following code samples demonstrates how to use a static content object to insert a title above the secondary collection.


{
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "prepend",
    "content": "<h2 class='bc-on-demand-section-title'>ON-DEMAND SESIONS</h2>",
    "require_roles": [],
    "omit_roles": []
}

Using the browser tools, you can see the h2 was inserted as the first element inside the bc-on-demand-section div.
