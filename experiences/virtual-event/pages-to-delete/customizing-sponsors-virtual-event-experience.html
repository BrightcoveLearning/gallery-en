Customizing the Event Sponsors

{{ page.description }}

Sponsor images can be added to the Virtual Event homepage.

The sponsor information is customized by editing the sponsors object in the configuration file.

  • name - Sponsor name
  • image_url - Sponsor logo, the images will be resized to 100px wide
  • link_url - Destination URL for the image

The sponsors object in the configuration file also has parameters to control where in the homepage the sponsor information is displayed. For information on positioning the sponsors section, see Positioning Content on the Home Page.

Removing opacity and sizing logo images

By default, logo images are styled with an opacity level of 80%. The following CSS style can be used to set the opacity to 100%. The max-width property can be used to change the size of the images.


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
Edit the experience and add the CSS to the top of the Custom Header HTML. field.
