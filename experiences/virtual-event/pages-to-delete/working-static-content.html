Working with Static Content

{{ page.description }}

Static content objects such as a welcome message are used to inject custom HTML into the template. This sample hero banner was constructed using a combination of text and images and injected into the top of the page.

The Gallery Site Editor also provides the ability to add a welcome message and/or image to the home page. Adding a static content image to the config file should be used when you are using role-based access control to conditionally display different images to different groups of users.

Static content can be any HTML and can be injected anywhere in the template. Static content objects are added to the static_content object in the configuration file. For information on how to position static content on the page, see Positioning Content on the Home Page.
