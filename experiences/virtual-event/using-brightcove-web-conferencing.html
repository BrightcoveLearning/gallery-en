Using Brightcove Web Conferencing

{{ page.description }}

Brightcove Web Conferencing provides the ability to hold online meetings.

Setting up audio and camera

When entering a web conferencing session, the settings page will appear allowing you to configure your audio and camera settings.

  • Camera - Select the camera to use; after selecting a camera, the camera preview image will update
  • Video quality - Select the video quality for the web conference
  • Mic - Select the mic to use
  • Speaker - Select the speaker to use for audio output
Note: The settings can also be configured from inside the web conference interface once the conference has started.

After the settings are configured, click Ready to enter the conference.

Using the Brightcove Web Conferencing interface

The interface provides a set of controls for controlling audio and video for the meeting.

Enabling attendee cameras and mics

Presenters have the ability to enable and disable mics and cameras for attendees. When the presenter disables cameras and mics, all attendees will be muted and cameras will not be displayed. When the meeting begins, mics and cameras are enabled by default.

Viewing the meeting attendee list

Click the people icon ( ) to toggle the display of meeting attendees. The attendee list will display on the right side of the page. If the attendee camera is enabled, their image will be displayed.
