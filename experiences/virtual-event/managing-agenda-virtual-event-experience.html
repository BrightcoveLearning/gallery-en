Managing the Event Agenda

{{ page.description }}

The Agenda module provides the ability to enter the agenda items manually or by uploading an external .csv agenda file so that all of the agenda items don't have to be entered manually. For information on creating an agenda file for upload, see Creating an Event Agenda File.

Using the Gallery Site Editor, you can create or upload an event agenda using the Agenda module. To access the Agenda module, follow these steps.

  1. Click the experience name to open the Gallery Site Editor.
  2. In the left navigation, click SITE FEATURES > Agenda.

Creating a new agenda

To create a new agenda for the experience, click New agenda in the Agenda module.

The agenda Status will be DEACTIVATED which indicates that the agenda will not be displayed in the Virtual Event Experience. Click on the Event name to add tracks and sessions.

Uploading an agenda file

Before importing your existing agenda into the Agenda module, the agenda file should be exported to a .csv file. The Agenda module can only import a .csv file. To upload an agenda file to a Virtual Event Experience, follow these steps.

  1. Click Upload Agenda.
  2. Browse and select the agenda file to upload.
  3. Confirm the agenda is listed on the My agenda page.

If any errors occur during the agenda import, a message will be displayed. After the import, the agenda Status will be DEACTIVATED which indicates that the agenda will not be displayed in the Virtual Event Experience.

Deleting the agenda

To delete the agenda, click the delete icon () next to the agenda.

Exporting the agenda file

The Agenda module provides the ability to download the agenda file. Downloading the agenda can be useful when:

  • You want to download an event agenda to use in another experience
  • An event planner wants to to download a copy of the agenda and maybe work with it in Microsoft Excel or another tool

To download the agenda, click the download icon () next to the agenda.

Editing the agenda title

The agenda title will be displayed on the experience homepage above the agenda. To edit the agenda title, click the edit icon, enter a title and then press enter.

Viewing the agenda

After an agenda file has been created, it will appear on the My agenda page. To view the agenda details, click on the Event name. The agenda will open displaying the details.

Click a date on the left side of the page to view the agenda for that day. The tracks are displayed as column headers at the top of the page.

Adding a session

To add a new session, follow these steps.

  1. Click on the agenda to open the agenda editor.
  2. Click on an open time slot or click Add Session at the top of the page .
  3. Enter the session information.
    • Session Name
    • Speakers - Click Select Speakers and select one or more speakers for the session
    • Start Date
    • Start Time
    • Duration - Session duration; for Interactive Room session types, the session will automatically end 5 minutes after the specified duration time
    • Description
    • Track - Select a track to assign the session to
    • Session Type
      • Session
      • Keynote
      • Meeting
      • Other
      • Interactive Room - Speaker email addresses are required so the interactive room can identify who should receive presenter permissions upon entering
    • Session Type Parameters - Based upon the Session Type that is selected, additional parameters will be displayed:
      • Video ID - Video ID of the video to display for this session (available with Session, Keynote and Other session types)
      • URL - Meeting URL to open (available with Meeting session type)
      • Pigeonhole Session ID and Pigeonhole Passcode - The Pigeonhole Session ID and Passcode are needed to display a Pigeonhole along with the breakout room (available with Breakout Room session type)
    • Cover Image - The cover image appears with the session in the agenda display
    • Related Content - Can be used to provide a link to session content, for example, handouts or slides
      • Label - Label for the content
      • URL - Link to the content
  4. Click Save.

Confirm the session is added to the agenda. If there are multiple sessions in the same track scheduled at the same time, they will be highlighted and appear next to each other.

Editing and deleting sessions

To edit or delete a session, click on a session to open the Edit Session panel. To edit the session, make any changes and click Save. To delete the session, click Delete Session.

Adding speakers to a session

To add speaker(s) to a session, follow these steps.

  1. Click on a session to open the session editor.
  2. Click Select Speakers.
  3. Select one or more speakers or click Create speaker to create a new speaker.
    Note: For information on creating speakers, see Managing Event Speakers.
  4. Click Save.
The Search field can be used to search for speakers.

Adding a new track

Tracks are used to group related sessions. For example, the Marketing track would have all the marketing related sessions. An agenda will typically have multiple tracks. To add a new track, follow these steps.

  1. Click on the agenda to open the agenda editor.
  2. Click the Add Track link at the top of the agenda. A new Unnamed Track will be created. You may have to scroll the agenda to view it.
  3. Click on the Unnamed Track to edit the name.
  4. Press enter to save the change.

Editing a track

To edit a track, click on the track name, make any changes and press enter to save the changes.

Reordering tracks

To change the order of tracks in the agenda, follow these steps.

  1. Click on the agenda to open the agenda editor.
  2. Hover over the track name to move, click the grab icon and drag the track to the new location.

Activating and deactivating an agenda

The agenda status determines whether or not the agenda will be displayed in the associated Virtual Event Experience. The status is displayed in the page header below the agenda name.

  • ACTIVATED - The agenda will be displayed
  • DEACTIVATED - The agenda will not be displayed

To activate an agenda, click Activate at the top of the page. To deactivate an agenda, click on it and click Deactivate. Note that after activating/deactivating an agenda, the experience doesn't need to be republished.
