Displaying Event Speakers on the Homepage

{{ page.description }}

The speaker information is customized by editing the speakers object in the configuration file.

  • name - Speaker name
  • description - Speaker information that displays below their image. This is limited to approximately 3 lines of 15 words.
  • image_url - Speaker image. Recommend a max width of 200px. Images will not be resized
  • link_url - (optional) Destination URL for the image (may link to speaker LinkedIn page for example)
Note: A maximum of 8 speakers can be displayed on the homepage.
Note: A maximum of 4 speakers can be displayed in a row.

By default, the template CSS configures the speaker images with a border-radius of 50% which will make the images circular.

The speakers object in the configuration file also has parameters to control where in the homepage the speaker information is displayed. For information on positioning the speakers section, see Positioning Content on the Home Page.
