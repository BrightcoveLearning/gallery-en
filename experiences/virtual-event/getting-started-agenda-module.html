Getting Started with the Agenda Module

{{ page.description }}

To access the Agenda module, click on a Virtual Event Experience to open the Gallery Site Editor and then click SITE FEATURES > Agenda.

From the home page you can:

  • Upload an event agenda
  • Manage the event agenda
  • Manage event speakers
To return to the Gallery Site Editor from the Agenda module, click the Back to My Event link on the left side of the page.

Uploading an event agenda

To initially create an event agenda, an agenda .csv file must be uploaded. For information on uploading an event agenda, see Uploading an agenda file.

Managing the event agenda

To edit the event agenda, click on it and make any changes as needed. For information, see Managing the Event Agenda.

Managing the event speakers

To manage the speakers associated with an event, click Speakers in the left navigation. For information on managing speakers, see Managing Event Speakers.
