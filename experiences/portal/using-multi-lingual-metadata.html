The multi-lingual metadata feature allows you to enter video metadata in multiple languages and then configure a Portal or In-Page Experience to display the video metadata in the same language that the experience is configured for.

Note: Multi-lingual metadata is not enabled by default. Contact your account manager if you would like to have it turned on for your account(s).

Configuring the account languages

Before you can add multi-lingual metadata to your videos, you must configure a primary language and one or more additional languages as part of the account settings. For information on how to do this, see Configuring Language Settings.

When configuring additional languages, many languages have multiple dialects to choose from. Gallery currently supports these languages and ISO codes.

Language ISO Code
English (US) en-us
German de-de
Spanish es-es
French fr-fr
Japanese ja-jp
Korean ko-kr
Russian ru-ru
Simplified Chinese zh-cn
Traditional Chinese zh-tw
Note: Make sure the ISO code(s) of the additional languages you add are listed in the above table or you will not see localized metadata in your experience.

Adding multi-lingual metadata

To add multi-lingual metadata to a video, follow these steps.

  1. Open the Media module.
  2. Click the video title link to open the video properties page.
  3. Select the language to add metadata for using the language selector.
    Note: Multi-lingual metadata cannot be added to videos that were transcoded using a Legacy ingest profile.
  4. Enter the localized value(s) and then click Save.

The following video property sections support multi-lingual metadata:

  • Video Information
  • Audio Tracks
  • Images
  • Custom Fields
  • Text Tracks
For more information, see Working with Multi-Lingual Metadata.

Setting the experience language

The Experience Language setting controls the Gallery specific text that is displayed on the experience.Â To configure the Experience Language, follow these steps.

  1. Open the Gallery module.
  2. Click on an experience title to open the Gallery Site Editor.
  3. In the left navigation, click Overview.
  4. Select the Experience Language.
  5. Click Save.

Enabling localized video metadata

In order to have the multi-lingual video metadata appear, the Language setting for the experience must be configured.

  1. Open the Gallery module.
  2. Click on an experience title to open the Gallery Site Editor.
  3. In the left navigation, click on
    • Portal Experiences - Click Site Features > Language
    • In-Page Experiences - Click Settings > Language
  4. Select the Use metadata's language as default option.
  5. Click Save.

Make sure you republish the experience.
