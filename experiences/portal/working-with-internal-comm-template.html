Video is an ideal way to distribute timely messages throughout an organization. The Internal Communications Experience is designed to help you deliver:

  • Company updates
  • Event briefs
  • Executive messages

The Internal Communications Experience is a portal template and provides most of the features found in the other Gallery portal templates. This topic will outline some of the differences in this template compared to other portal templates.

Note: To have the Internal Communications template enabled for your account, contact your Customer Success Manager.
Note: For a step-by-step guide to creating a Portal Experience, see Step-by-Step: Creating and Publishing a Portal Experience.

Creating a new experience

To create a new Internal Communications Portal Experience, follow these steps.

  1. In the primary navigation, clickÂ Gallery.
  2. Click New Experience.
  3. Click the Portal tab at the top of the page. A list of Portal templates will be displayed. Locate the Internal Communications template and click Choose.
    Note: Once you select a template it can't be changed. For more information Gallery templates, see the Overview of Gallery Portal Templates topic.
  4. Enter an Experience Title.
  5. Click Create Experience.

The Gallery Site Editor will open. The process to add videos and customize the experience is similar to the other portal templates.

Video collection organization

The order of the video collections controls how they appear on the homepage.

Note: For information on creating collections and adding videos to a collection, see Adding Videos to a Portal Experience.

The Internal Communications template has four main areas where videos are displayed as designated in the sample site:

  1. Featured collection - Typically used to feature the live or simulive content that will be displayed throughout the event. This should ideally be ordered based on its go-live time from earliest to latest for the best experience.
  2. Primary collection - Can include anything but is typically used to feature specific VOD breakouts or content you want to drive additional attention to
  3. Secondary collection - Can include anything but is typically used to feature specific VOD breakouts or content you want to drive additional attention to
  4. Tracks - Typically used for VOD content
Note: The Featured, Primary and Secondary collections support a maximum of 24 videos.
Note: To insert a title above the primary/secondary collection, add a static content object to the configuration file. For information, see Working with Static Content.

When ordering collections in Gallery, the collections should be ordered in the following manner:

  • Collection 1 - Featured collection
  • Collection 2 - Primary collection
  • Collection 3 - Secondary collection
  • Collection 4 - Tracks

Adding a site menu

The Internal Communications template offers the ability to add a site menu to the top of the site pages. Menu links can be configure to go to specific videos collections or to external links. Follow these steps to add a menu.

  1. Open the Gallery module.
  2. Click on an Internal Communications experience to edit it.
  3. Click Add menu item at the top of the page.

Adding a menu item for a collection

Following these steps to add a menu item to link to a video collection.

  1. Click the Collection link.
  2. Enter a Label.
  3. Select the Collection that the menu link will go to.
  4. Select the Order for the menu item. The order controls the order in which the menu items are listed.
  5. Choose to Open in the same tab or Open in a new tab.
  6. Click Save.

Adding a menu item for a custom link

Following these steps to add a menu item to link to a custom link.

  1. Click the Custom link link.
  2. Enter the URL.
  3. Select the Order for the menu item. The order controls the order in which the menu items are listed.
  4. Choose to Open in the same tab or Open in a new tab.
  5. Click Save.

Editing or deleting a menu item

Following these steps to edit or delete an existing menu item.

  1. Hover over a menu item and click on the action menu (...).
  2. To delete a menu item, click Delete and then confirm the deletion.
  3. To edit a menu item, click Edit menu link, make the appropriate changes and then click Save.

Portal settings currently not supported

The following settings and options that are normally supported by portal templates are not currently supported in the Internal Communications template.

  • Pre-populate experience
  • Site search
  • Secondary font
  • CTAs
  • Site language
  • Social sharing
  • Video sorting in collection
  • Video downloads
