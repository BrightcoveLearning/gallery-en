Downloading Videos from a Portal Experience

{{ page.description }}
 

Portal Experiences can be configured to display a download link allowing viewers to download video content. Sites can be configured to provide a download link for all videos or specific videos based upon a custom field value. When a video is downloadable, a download link will appear next to the video. The highest quality MP4 rendition available is downloaded.

Note: The video must have at least one MP4 rendition available.
Note: YouTube videos can't be downloaded.

To provide a download link for all videos, see Customizing the Video Settings for a Portal Experience. To add download video support to a Portal Experience, you need to perform the following steps:

  1. Create a custom field to indicate downloadable content
  2. Tag video content that can be downloaded
  3. Configure a Portal Experience to allow downloads

Creating a custom field to indicate downloadable content

Videos that can be downloaded are indicated by setting the value of a custom field. Start by creating a custom field in your account that will be used to indicate that a video can be downloaded.

  1. Login to Video Cloud Studio.
  2. Click Admin and then Video Fields. Note that you must be an account administrator to access to the Admin menu.
  3. Click Add Custom Field.
  4. Create a new custom field that will be used to indicate whether a video can be downloaded or not. The name and value assigned do not matter. In the example below, a new custom field downloadable is being created. The Type is set to List, with 2 values, True and False. Videos that can be downloaded will have this value set to True as part of the video properties.
    Note: For information on creating custom fields, see Creating Custom Metadata Fields.
  5. Click Save Field.

Tag video content that can be downloaded

Any video that can be downloaded should have a value assigned to the custom field that was created.

  1. Open the Media module.
  2. To mark a video as downloadable, click on the video name to open the video properties.
  3. Locate the Custom Fields section and click Edit.
  4. Set the value of the custom field as appropriate. In this example, the value of Downloadable is set to True.
  5. Click Save.
  6. Edit the custom field value for any additional videos as required.

Configuring a Portal Experience to allow downloads

To configure a Portal Experience to allow video downloads, follow these steps.

  1. Open the Gallery module.
  2. Click on an experience title to edit it.
  3. Click Video and Playback > Video in the left navigation.
  4. To indicate that videos to be downloaded are based upon a custom field, in the Video Download section, select the Provide a download link based on a custom field value option.
  5. Enter the custom field name and the value that were created earlier. In this example, the custom field created was named downloadable and the value was set to true for videos that can be downloaded.
  6. Click Save and then republish the experience.
  7. Confirm that a Download Video link appears with the video(s).
