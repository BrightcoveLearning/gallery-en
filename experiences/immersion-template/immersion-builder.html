Overview

The Immersion template has added new capabilities, such as creating new pages and adding them to the portal. This allows to have complete and dynamic widgets on the pages. To learn more about these new capabilities, read the following sections.

Immersion Experience Capabilities

  1. First you must to create an Immersion Experience. To do it, follow the steps here: Creating a new experience.
  2. After you create the experience, click + Add a Page .
  3. Click Add a section.
  4. Notice how the Add section appears. In content, you can add Headlines and Texts. In widgets you can add Agenda and People.

    5. To learn more about Widgets, see Agenda Widget on the Immersion Gallery template and People Widget on the Immersion Gallery template. This document is focused on the Immersion build and the Content.

Headline

    To add a Headline, follow these steps:

  1. In Add Section, click Headline.
    • a. - Add the name you want to add
    • b.- Select whether you prefer title, subtitle, title section, subtitle section or mini subtitle.
    • c.- Click Save.

Text

    To add a text, follow these steps:

  1. In the Add section, click Text.
  2. Add the text and customize it with your preferred Font Size and Line Height.

  3. Click Save.
    4. Note: With the help of the arrow, you can move the text up or down.
