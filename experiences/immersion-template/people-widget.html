People Widget on the Immersion Gallery template

The people widget lets you see which people participate in the events you create. Its purpose is to simplify who participates in each event.

Create a Widget

First, you must create a custom page on an Immersion Experience to use an Agenda Widget implementation. Follow these steps:

  1. In the left navigation, click Gallery.

  2. Choose Immersion.
  3. In Experience Overview, customize your experience. Add a description and select the experience language. Then, click Create Experience.
  4. Create a custom page. Click + Add a Page and configure the page name and path.
  5. Click People.
  6. Navigate to People section.

If you haven't added any people yet:

  1. Click the Select + button.
  2. Notice how the start-creating people section appears. Click Create people.
  3. It automatically sends you to the Agenda module. For more information, see Managing Event Speakers.

If you already have added people:

  1. Notice the dropdown menu with selectable checkboxes appears. Select as many people as you prefer, then click Save.

Deleting People

Click the people widget, and then click Delete.
