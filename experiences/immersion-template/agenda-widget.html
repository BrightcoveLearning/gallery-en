Agenda Widget on the Immersion Gallery template

{{ page.description }}

Create a Widget

First, you must create a custom page on an Immersion Experience to use an Agenda Widget implementation. Follow these steps:

  1. In the left navigation, click Gallery.

  2. Choose Immersion.
  3. In Experience Overview, customize your experience. Add a description and select the experience language. Then, click Create Experience.
  4. Create a custom page. Click + Add a Page and configure the page name and path.
  5. Once in the Custom Page, use the Add a Section button to add an Agenda Widget.
  6. Click Agenda.

  7. In the Agenda, you can select previously created Agendas or create a new one to edit. You must create your first Agenda the first time you use it.
  8. Once in the Create Agenda section, the Agenda Module shows up. Here, you can create your Agendas and configure them. You are automatically taken to a brand new Agenda for you to configure. To learn more about the Agenda Module, please see Agenda and Speakers.

Edit a Widget

Once you created an Agenda, follow these steps to Edit a Widget:

  • Click on the Widget you want to edit. The controls for the said Widget will pop up.

  • Click Edit.

    • Deleting a Widget

    1. Select the Widget you want to delete.
    2. Click Delete.

    Notes:
    1. You can create as many Widgets as you want on your custom page
    2. All Widgets can source the same Agenda from the Agenda module
    3. Deleting a Widget does not delete its Agenda. All created Agendas can still be accessed from the Select Agenda Modal.
