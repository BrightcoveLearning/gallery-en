Pigeonhole can be used to easily add event based interactivity (live Q&A, Reactions, polls, surveys, etc.) to a Live Event In-Page Experience. Typically, a new Pigeonhole is created for the event and then sessions are created for the live events. Once the session is created, Pigeonhole embed code is pasted into the Media asset you want to put on. The embed code will cause the Pigeonhole Reactions display on the live event player.

Creating a Pigeonhole event and session

You should create a new Pigeonhole for your live event and then add sessions for each video that will have Pigeonhole Reactions enabled.

  1. Login to your Pigeonhole account.
  2. Click + Add Pigeonhole and click the account to use.
  3. Enter an Event name and Passcode.
  4. Enter Start and End dates for the event. These dates control how long the Pigeonhole will be active.
  5. Select a Timezone and Custom Branding option (if applicable).
  6. Click Continue.
  7. Now that the event is created, the Agenda item will be selected in the left navigation. Click Add session > Create new.
  8. Select a Session Type of Chat.
  9. Enter a Session name.
  10. Configure the Chat Settings as appropriate.
  11. Click Add Chat. Confirm the session was created.

Generating the Pigeonhole Reactions embed code

In order for the Pigeonhole to display in the experience, the following code will be added using a custom field on a Video Cloud video asset.

To generate the code for the Pigeonhole Reactions, follow these steps.

  1. In Pigeonhole, click More settings in the left navigation.
  2. On the Reactions tab, click the dropdown arrow For external video integration.
  3. Create reactions widget
  4. Select the Chat sesion we just created.
  5. Create widget.
  6. Copy embed code.
  7. Copy the embed code.
  8. close.

Adding Pigeonhole Reactions to a Media asset

Adding a Pigeonhole to a media asset is done by using a Custom field. To add a Pigeonhole reactions, follow these steps.

  1. Open the Media module.
  2. Choose the video in which you want to embed Pigeonhole Reactions.
  3. On Custom fields click Edit.
  4. Paste the Pigeonhole code without the scrip tags in the Pigeonhole Reactions field.
  5. Save

Now, you can add the video asset to any templates supporting pigeonhole reactions pigeonhole_reations custom field (Immersion template, Virtual event, Live portal, or Live Event In-Page) in Gallery.
