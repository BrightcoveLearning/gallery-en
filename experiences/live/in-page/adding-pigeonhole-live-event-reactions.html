Adding Pigeonhole Reactions to a Gallery Experience

{{ page.description }}

Pigeonhole can be used to easily add event based interactivity (live Q&A, Reactions, polls, surveys, etc.) to a Gallery Experience. Typically, a new Pigeonhole is created for the event and then sessions are created for the live events. Once the session is created, Pigeonhole embed code is pasted into the Media asset you want to put on. The embed code will cause the Pigeonhole Reactions display on the event player.

Creating a Pigeonhole event and session

You should create a new Pigeonhole for your live event and then add sessions for each video that will have Pigeonhole Reactions enabled.

  1. Login to your Pigeonhole account.
  2. Click + Add Pigeonhole and click the account to use.
  3. On Event type select Virtual
  4. Enter an Event name and Passcode.
  5. Enter Start and End dates for the event. These dates control how long the Pigeonhole will be active.
  6. Select a Timezone and Expected total participants.
  7. Click Continue.
  8. Now that the event is created, the Agenda item will be selected in the left navigation. Click Add session > Create new.
  9. Select a Session Type of Chat.
  10. Enter a Session name.
  11. Configure the Chat Settings as appropriate.
  12. Click Add Chat. Confirm the session was created.

Generating the Pigeonhole Reactions embed code

In order for the Pigeonhole to display in the experience, the following code will be added using a custom field on a Video Cloud video asset.

To generate the code for the Pigeonhole Reactions, follow these steps.

    1. In Pigeonhole, click More settings in the left navigation.
    2. Reactions.
    3. click the dropdown arrow For external video integration.
  2. Create reactions widget
  3. Select the Chat sesion you created.
  4. Create widget.
  5. Copy embed code.
  6. Copy the Pigeonhole code.
  7. close.

Adding Pigeonhole Reactions to a Media asset

Adding Pigeonhole Reactions to a Video Cloud Media asset is done by using the pigeonhole_reactions custom field. To add Pigeonhole reactions to a Media asset, follow these steps.

  1. Open the Media module.
  2. Choose the video in which you want to embed Pigeonhole Reactions.
  3. On Custom fields click Edit.
  4. Paste the Pigeonhole code in the Pigeonhole Reactions field.
    Make sure to strip out the script tags <script type="text/javascript"> at the beginning of the Pigeonhole embed code and </script> at the end, so that the code would end up like the following snippet:

    (function(p,i,g,e,o,n,s){p[o]=p[o]||function(){(p[o].q=p[o].q||[]).push(arguments)}, n=i.createElement(g),s=i.getElementsByTagName(g)[0];n.async=1;n.src=e; s.parentNode.insertBefore(n,s);}) (window,document,'script','https://static.pigeonhole.at/widget/pigeon-widget.js','phl'); phl("create", { width: "100%", height: "100%", className: "pigeonhole-reactions-iframe", reactions: true });

  5. Save

Create Pigeonhole custom field

You must be an account administrator to access the ADMIN menu.
    1. On Video Cloud go to Admin
    2. Video Fields
  2. Add Custom Field.
  3. Create the pigeonhole_reactions custom field.
  4. Save Field
