Gallery Template Release Notes

{{ page.description }}

Over time, new features and enhancements may be made to templates by Brightcove. Brightcove will not push major template updates out to exiting sites. Publishers are responsible for upgrading their sites to use new templates and then for testing sites before publishing. For information on updating a Gallery site to use a new template version, see Updating Gallery Templates.

Changes made to templates are documented as part of the template version release notes. Release notes are available for the following templates:

  • Catalogue
  • Chronicle
  • Classic
  • Live Event
  • Marquee
  • Mosaic
  • Publisher
  • Showcase

Catalogue release notes

v5.0 - Feb 14, 2018

  • Added optional title to homepage

v4.0 - Jan 24, 2018

  • Updated carousel to center video on mobile devices when collection contains only one video
  • Made carousel transitions cleaner

Chronicle release notes

v3.0 - Dec 22, 2016

  • Improved thumbnail images using the new image loader

v2.0 - Oct 31, 2016

  • Added two layout options for hero video
    • The new layout has a smaller hero image for publishers who don't have high-quality thumbnail images
  • Removed pagination in favor of lazy-loading on scroll
  • Added the CTA-support for lazy-loaded videos
    • Similar to showcase Chronicle has the option of repeating the CTA after 5,10, and 15 videos
  • Updated collection navigation / menu layout
    • Replaced the full-screen navigation with a navigation bar on the left side of the screen
